This directory should contain all the images used on the website. Below is a list of required images:

## Image Optimization Tips

1. **Format**: Use WebP format where possible for better compression
2. **Size**: Compress all images to reduce load time (use tools like TinyPNG, ImageOptim)
3. **Alt Text**: Always include descriptive alt text for SEO
4. **Lazy Loading**: Images are set to lazy load in the code
5. **Responsive**: Provide multiple sizes for different screen sizes

## Where to Get Images

1. **Professional Photography**: Hire a photographer to take pictures of your actual fleet
2. **Stock Photos** (temporary): 
   - [Unsplash](https://unsplash.com) - Free high-quality images
   - [Pexels](https://pexels.com) - Free stock photos
   - [Pixabay](https://pixabay.com) - Free images
3. **Your Own Fleet**: Take professional photos of your actual vehicles

## Image Specifications

### Hero Background
- **Type**: Taxi/cars on road, professional service
- **Mood**: Professional, trustworthy, modern
- **Colors**: Complementary to website color scheme (#ff6b35, #004e89)

### Car Images
- **Angle**: Side view or 3/4 angle
- **Background**: Clean, preferably white or light gray
- **Lighting**: Well-lit, professional
- **Branding**: Can include your company branding

### User Testimonials
- **Style**: Professional headshots or happy customers
- **Background**: Neutral or blurred
- **Expression**: Friendly, satisfied

## Favicon
Create a favicon from your logo:
- `favicon.ico` - 32x32px icon file
- `favicon-16x16.png` - 16x16px
- `favicon-32x32.png` - 32x32px
- `apple-touch-icon.png` - 180x180px for iOS devices

## File Naming Convention
- Use lowercase
- Use hyphens instead of spaces
- Be descriptive
- Include relevant keywords for SEO

Example: `ranchi-taxi-swift-dzire.jpg` instead of `IMG001.jpg`
