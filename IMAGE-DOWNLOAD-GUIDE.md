## üìã Implementation Steps

### Step 1: Download Images
1. Use Google Images with search terms provided above
2. Filter: **Tools ‚Üí Size ‚Üí Large** (at least 1280px wide)
3. Filter: **Usage Rights ‚Üí Creative Commons licenses** (or use stock photos)
4. Download and save with exact filenames mentioned above
5. Rename downloaded images to match naming convention

### Step 2: Optimize Images
Before placing in folders:
```bash
# Use online tools or:
- TinyPNG.com (compress without quality loss)
- Squoosh.app (Google's image optimizer)
- Target size: 100-300KB per image
- Format: JPG for photos, WebP for better compression
```

### Step 3: Place in Correct Folders
Move images to respective folders:
- All city images ‚Üí `images/cities/`
- All waterfalls ‚Üí `images/waterfalls/`
- Hill stations ‚Üí `images/hill-stations/`
- Major destinations (Bodh Gaya, Varanasi, Digha, Betla) ‚Üí `images/destinations/`
- Services ‚Üí `images/services/`
- Vehicles ‚Üí `images/vehicles/`

### Step 4: Update HTML Files
Images will be automatically referenced in enhanced HTML sections with proper lazy loading.

## üí° Pro Tips

1. **Aspect Ratio:** Prefer 4:3 or 16:9 for better display
2. **Consistency:** All images in a category should have similar tone/quality
3. **Seasonal:** Get monsoon images for waterfalls (July-Sept)
4. **Time of Day:** Sunrise/sunset images more attractive
5. **People:** Include people for scale and relatability (not too crowded)
6. **Landmarks:** Focus on iconic/recognizable landmarks
7. **Alt Text:** Each image will have proper alt text in HTML for SEO
